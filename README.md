# baileys-n8n

To install dependencies:

```bash
npm install
```

To run:

```bash
npm run start
```

To run in development mode:

```bash
npm run dev
```

Default server routes:

- `GET /`
- `GET /health`

Default server port:

```bash
3000
```

You can override it with:

```bash
PORT=4000 npm run start
```

This project now runs on [Node.js](https://nodejs.org) with [Express](https://expressjs.com).
